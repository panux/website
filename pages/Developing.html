Before you start building packages, you will need to install Docker (for Linux) on your system.

Next, git clone the packages-main repository (https://github.com/panux/packages-main.git). Then, cd into it.

To build a package, use make. The make target is the directory the package is in and the name of the package without a file extension seperated by a slash. For example, to build bash you would run: make main/bash. The resulting packages will be placed in a subdirectory called out. While the extension of these files is .tar.xz, they are actually uncompressed.

To read about how to create new package generator files, read Package Generator
