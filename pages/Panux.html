What is Project Panux?

Project Panux is a project to develop open-source software, mainly in the form of a Linux distro.

Panux is a lightweight Linux distro built from scratch. The goal of Panux is to be as customizable but easy-to-use as possible. It is currently in beta.

What is different between Panux and other Linux distros?

Panux is designed to be lightweight, but also uses a simple system to generate packages. This system is relatively fast, and runs in Docker containers so that the build can be run on any machine.

How can I get started with Panux?

Unfortunately, the bootloader setup is still work in progress, so you cannot install Panux directly. However, Panux works very well in containers - so you can use it with Docker. There is also a Live Demo on our website using Docker containers.

What distro is Panux based off of?

Panux is a distro built from scratch - it is not based off of any other distro.
