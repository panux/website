Before you start building packages, you will need to install Docker and Go (for Linux) on your system.

You will need the pkgen tool, which can be installed with go get github.com/panux/pkgen

In addition, to build the Docker containers used to build packages, you will need to git clone https://github/panux/panux-builder.git and run the compile.sh script as root (or with Docker permission).

Next, git clone the packages-main repository (https://github.com/panux/packages-main.git). Then, cd into it.

To build a package, use make. The make target is the name of the package. For example, to build bash you would run: make bash. The resulting packages will be placed in a subdirectory called out/out (configurable via environment variable).

To read about how to create new package generator files, read Package Generator
